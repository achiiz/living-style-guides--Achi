What is a living style guide

A living style guide is an important tool for designers and developers to understand how an application will look and feel. It remains one of the best ways to communicate design standards to an organization, especially in large, complex web applications. Living style guides contain powerful functions such as colours schemes, common styles and UX patterns which help both development and design move at a faster pace.

Why is a living style guide useful to developers and designers

A living style guide keeps designers and developers in sync, and transforms Front-end codebase into well-described pattern libraries with minimal efforts. It contains powerful functions such as colours schemes, common styles and UX patterns which help both development and design move at a faster pace.

My list of living style guide tips

  • Start with basics like colour and typography
